The mlld SDK embeds the interpreter in Node.js applications and provides language bindings for Go, Python, Rust, Ruby, and Elixir. Core capabilities: file-based execution with structured results, string evaluation, dynamic module injection, static analysis, in-flight state updates and event streaming via async handles, MCP server injection, filesystem integrity (signing and verification), and security label helpers. `VirtualFS` adds copy-on-write file overlays for sandboxed workflows (JS/TS only).
